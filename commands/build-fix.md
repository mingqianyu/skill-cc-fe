# /build-fix 命令

增量修复 TypeScript 和构建错误。

## 执行流程

1. 运行构建: `pnpm build` 或 `npm run build`
2. 解析错误输出，按文件分组、按严重程度排序
3. 对每个错误：
   - 展示错误上下文（前后 5 行）
   - 解释问题
   - 提出修复方案
   - 应用修复
   - 重新构建验证
4. 停止条件：
   - 修复引入新错误
   - 同一错误 3 次未解决
   - 用户要求暂停
5. 展示摘要：已修复 / 剩余 / 新增

**一次修一个错误，安全第一！**
