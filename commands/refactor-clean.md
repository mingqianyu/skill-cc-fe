# /refactor-clean 命令

安全识别和移除死代码。

## 执行流程

1. 运行死代码分析
2. 按安全级别分类：
   - **SAFE**: 未使用的工具函数、测试文件
   - **CAUTION**: API 路由、组件
   - **DANGER**: 配置文件、入口文件
3. 只提议安全的删除
4. 每次删除前运行测试
5. 删除后再次运行测试
6. 测试失败则回滚
7. 展示清理摘要

**绝不在未运行测试的情况下删除代码！**
